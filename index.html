Dicionário de Tags HTML

HTML IMG UL e LI OL e LI BODY DIV FOOTER FORM INPUT SMALL TABLE STRONG

1.<HTML>

Define o início e fim de um documento HTML



2.<IMG>

Permite adicionar uma imagem e sua mensagem alternativa caso ela não possa ser exibida



3.<UL e LI>

Cria uma lista não-ordenada e seus itens (UL = unordered list, LI = list item)



4.<OL e LI>

Cria uma lista ordenada e seus itens (OL = ordered list, LI = list item)



5.<BODY>

Define o corpo do documento, nele ficam os conteúdos do documento HTML como headings, paragrafos, imagens, hyperlinks, tables, lists, etc.



6.<DIV>

Define uma seção ou divisão no documento HTML, comumente usado como container para ser trabalhado com CSS



7.<FOOTER>

Define o rodapé do documento HTML



8.<FORM>

Cria um formulário para input do usuário



9.<INPUT>

Cria um campo para input do usuário



10.<SMALL>

Transforma o texto em pequeno



11.<TABLE>

Cria uma tabela



12.<STRONG>

Define o texto como bold
